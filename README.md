# Earth Satellite Simulator

A Python-based Earth satellite simulator that visualizes orbital motion using mathematical modeling and real-time animation.

## Features
- Earth visualization
- Orbit visualization
- Satellite motion simulation
- Real-time animation with matplotlib

## Technologies
- Python
- NumPy
- Matplotlib

## How to run

Install dependencies:

    pip install numpy matplotlib

Run the project:

    python main.py
